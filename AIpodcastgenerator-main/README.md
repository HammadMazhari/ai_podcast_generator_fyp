## Setup 

1. Install Python 3.9 or later.
    
    
2. Go to the server folder
    ```bash

    cd server
    ```


 3. Install the required Python packages:

    ```bash
    pip install -r requirements.txt
    ```

4. Run the script:

    ```bash
    python app.py
    ```


Open another terminal windows
    
5. 
    ```bash

    cd client
    ```


6. install packages
    ```bash

   npm i
    ```


7. start the client
    ```bash

   npm start

    ```
